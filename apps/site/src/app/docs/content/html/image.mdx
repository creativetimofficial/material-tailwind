# Tailwind CSS Image

Our Tailwind CSS Image component simplifies image handling and styling, offering an array of classes to achieve the image effects that you are looking for in your website or application.

From adjusting image sizes to creating visually appealing galleries, Tailwind CSS empowers you to optimize and customize your web images with ease.

See below our examples of image UI components.

---

## Image Demo

This example displays a simple image within a scrollable, centered grid layout. The responsive design classes ensure that the layout adapts to different screen sizes, enhancing the user experience across devices.

<OldComponentPreview demo={<HTMLImage.ImageDemo />}>
```html
<img
  src="https://dub.sh/Y0NxRWv"
  alt="nature-image"
  class="object-cover object-center w-full h-96"
/>
```
</OldComponentPreview>

## Circular Image

Use `rounded-full` class to round the corners of the image into a circle. This effect will be most noticeable if the image is square; otherwise, the image will look elliptical.

<OldComponentPreview demo={<HTMLImage.CircularImage />}>
```html
<img
  src="https://dub.sh/Y0NxRWv"
  alt="nature-image"
  class="object-cover object-center rounded-full h-96 w-96"
/>
```
</OldComponentPreview>

## Image with Shadow

In this example, the `shadow-xl shadow-black/25` classes add a large shadow effect with a black color and opacity (25%). This creates a sense of depth and improves the image visually from its background, making it stand out more prominently.

<OldComponentPreview demo={<HTMLImage.ImageWithShadow />}>
```html
<img
  src="https://dub.sh/Y0NxRWv"
  alt="nature-image"
  class="h-96 w-full rounded-lg object-cover object-center shadow-lg shadow-black/25"
/>
```
</OldComponentPreview>

## Image with Caption

Use this example if you want to include a caption below the image. Adding images with captions enriches the content and provides visual breaks in the text.

<OldComponentPreview demo={<HTMLImage.ImageWithCaption />}>
```html
<figure>
  <img
    src="https://dub.sh/Y0NxRWv"
    alt="nature-image"
    class="object-cover object-center w-full rounded-lg h-96"
  />
  <caption class="block mt-2 font-sans text-sm antialiased text-center text-inherit">
    Image Caption
  </caption>
</figure>
```
</OldComponentPreview>

## Image with Blurred Caption

In the example below, the styling includes a semi-transparent white background with a blur effect (`backdrop-blur-md`), making the text readable regardless of the image behind it. The border, padding, and shadow improve visibility and aesthetic appeal.

<OldComponentPreview demo={<HTMLImage.ImageWithBlurredCaption />}>
```html
<figure class="relative w-full h-96">
  <img
    src="https://dub.sh/Y0NxRWv"
    alt="nature-image"
    class="object-cover object-center w-full h-full rounded-xl"
  />
  <figcaption class="absolute bottom-4 left-1/2 w-[calc(100%-2rem)] -translate-x-1/2 overflow-hidden rounded-lg border border-surface bg-background bg-opacity-75 shadow-lg shadow-black/5 saturate-200 backdrop-blur-md">
    <div class="flex justify-between w-full px-4 py-2 rounded h-max">
      <div>
        <h6 class="font-sans text-base antialiased font-bold text-inherit md:text-lg lg:text-xl">
          Sara Lamalo
        </h6>
        <p class="mt-1 font-sans text-base antialiased text-foreground">
          20 July 2022
        </p>
      </div>
      <p class="font-sans text-base antialiased font-bold text-inherit">
        Growth
      </p>
    </div>
  </figcaption>
</figure>
```
</OldComponentPreview>
