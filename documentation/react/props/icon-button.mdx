# Tailwind CSS Icon Button - Props

<span id="icon-button-props" className="scroll-mt-48" />
The following props are available for icon button component. These are the custom
props that come with we've added for the icon button component and you can use all
the other native button props as well.

| Attribute   | Type                      | Description                          | Default                                |
| ----------- | ------------------------- | ------------------------------------ | -------------------------------------- |
| `variant`   | [Variant](#types-variant) | Change icon button variant           | <Code>filled</Code>                    |
| `size`      | [Size](#types-size)       | Change icon button size              | <Code>md</Code>                        |
| `color`     | [Color](#types-color)     | Change icon button color             | <Code>blue</Code>                      |
| `ripple`    | <Code>boolean</Code>      | Add ripple effect for icon button    | <Code>true</Code>                      |
| `className` | <Code>string</Code>       | Add custom className for icon button | `''`                                   |
| `children`  | <Code>node</Code>         | Add content for icon button          | No default value it's a required prop. |

<br />
<br />

### For TypeScript Only

```tsx
import type { IconButtonProps } from "@material-tailwind/react";
```
