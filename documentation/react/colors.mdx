<DocsTitle href="customizing-the-default">
## Customizing The Default Color Palette
</DocsTitle>

<span id="customizing-the-default" className="scroll-mt-48" />

Customizing colors for @material-tailwind/react is very easy and straightforward, it's the same as tailwindcss colors customization.

You just need to modify the <Code>colors</Code> object for the <Code>tailwind.config.js</Code> file.

```js
module.exports = withMT({
  theme: {
    colors: {
      // Configure your color palette here
    },
  },
});
```

<DocsTitle href="adding-new-color">
## Adding New Color
</DocsTitle>

<span id="adding-new-color" className="scroll-mt-48" />

You can add new color to @material-tailwind/react color palette very easy and straightforward, it's the same as adding new color for tailwindcss.

You just need to add your own color to the <Code>extend</Code> and <Code>colors</Code> object for <Code>tailwind.config.js</Code> file.

```js {5-16}
module.exports = withMT({
  theme: {
    extend: {
      colors: {
        sky: {
          50: "#f0f9ff",
          100: "#e0f2fe",
          200: "#bae6fd",
          300: "#7dd3fc",
          400: "#38bdf8",
          500: "#0ea5e9",
          600: "#0284c7",
          700: "#0369a1",
          800: "#075985",
          900: "#0c4a6e",
        },
      },
    },
  },
});
```
