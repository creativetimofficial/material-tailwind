<DocsTitle href="avatar">
# Tailwind CSS Avatar - React
</DocsTitle>

Use our Tailwind CSS <Code>Avatar</Code> component to portray people or objects in your web projects. The <Code>Avatar</Code> can be used as a visual identifier for a user profile on your website.

See below our beautiful <Code>Avatar</Code> example that you can use in your Tailwind CSS and React project. The example also comes in different styles and colors so you can adapt it easily to your needs.

<br />

<CodePreview link="avatar#avatar" component={<AvatarExamples.AvatarDefault />}>
```jsx
import { Avatar } from "@material-tailwind/react";

export function AvatarDefault() {
  return <Avatar src="/img/face-2.jpg" alt="avatar" />;
}
```
</CodePreview>

---

<DocsTitle href="avatar-variants">
## Avatar Variants
</DocsTitle>

The <Code>Avatar</Code> component comes with 3 different variants that you can change it using the <Code>variant</Code> prop.

<CodePreview link="avatar#avatar-variants" component={<AvatarExamples.AvatarVariants />}>
```jsx
import { Avatar } from "@material-tailwind/react";

export function AvatarVariants() {
  return (
    <div className="flex gap-4">
      <Avatar src="/img/face-2.jpg" alt="avatar" />
      <Avatar src="/img/face-2.jpg" alt="avatar" variant="rounded" />
      <Avatar src="/img/face-2.jpg" alt="avatar" variant="square" />
    </div>
  );
}
```
</CodePreview>

<DocsTitle href="avatar-with-border">
## Avatar with border
</DocsTitle>

You can add border around the avatar using the <Code>withBorder</Code> prop. To change the color of the avatar border use the <Code>color</Code> prop, by default it's <Code>blue</Code>.

<CodePreview component={<AvatarExamples.AvatarWithBorder />}>
```jsx
import { Avatar } from "@material-tailwind/react";

export function AvatarWithBorder() {
  return (
    <div className="flex gap-4">
      <Avatar
        src="/img/face-2.jpg"
        alt="avatar"
        withBorder={true}
        className="p-0.5"
      />
      <Avatar
        src="/img/face-2.jpg"
        alt="avatar"
        variant="rounded"
        withBorder={true}
        color="green"
        className="p-0.5"
      />
      <Avatar
        src="/img/face-2.jpg"
        alt="avatar"
        variant="square"
        withBorder={true}
        color="pink"
        className="p-0.5"
      />
    </div>
  );
}
```
</CodePreview>

<DocsTitle href="avatar-props">
## Avatar Props
</DocsTitle>

The following props are available for avatar component. These are the custom props that we've added for
the avatar component and you can use all the other native img props as well.

| Attribute    | Type                      | Description                      | Default               |
| ------------ | ------------------------- | -------------------------------- | --------------------- |
| `variant`    | [Variant](#types-variant) | Change avatar variant            | <Code>circular</Code> |
| `size`       | [Size](#types-size)       | Change avatar size               | <Code>md</Code>       |
| `color`      | [Color](#types-color)     | Change avatar border color       | <Code>blue</Code>     |
| `className`  | <Code>string</Code>       | Add custom className for avatar  | `''`                  |
| `withBorder` | <Code>boolean</Code>      | Add 2px border around the avatar | <Code>false</Code>    |

<br />
<br />

### For TypeScript Only

```tsx
import type { AvatarProps } from "@material-tailwind/react";
```

<DocsTitle href="avatar-theme">
## Avatar Theme
</DocsTitle>

Learn how to customize the theme and styles for avatar component, the theme object for avatar component has three main objects:

**A.** The <Code>defaultProps</Code> object for setting up the default value for props of avatar component.<br />
**B.** The <Code>valid</Code> object for customizing the valid values for avatar component props.<br />
**C.** The <Code>styles</Code> object for customizing the theme and styles of avatar component.<br />

You can customize the theme and styles of avatar component by adding Tailwind CSS classes as key paired values for objects.

<br />
<br />

<DocsTitle href="avatar-theme-object-type">
## Avatar Theme Object Type
</DocsTitle>

```ts
interface AvatarStyleTypes {
  defaultProps: {
    variant: string;
    size: string;
    className: string;
    withBorder: boolean;
    color: string;
  };
  valid: {
    variants: string[];
    sizes: string[];
    colors: string[];
  };
  styles: {
    base: {
      initial: object;
      withBorder: object;
    };
    sizes: {
      xs: object;
      sm: object;
      md: object;
      lg: object;
      xl: object;
      xxl: object;
    };
    variants: {
      square: object;
      rounded: object;
      circular: object;
    };
    borderColor: object;
  };
}
```

<br />
<br />

### For TypeScript Only

import type { AvatarStyleTypes } from "@material-tailwind/react";
