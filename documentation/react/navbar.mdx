# Tailwind CSS Navbar - React
</DocsTitle>

Use our responsive Tailwind CSS navigation for your website. You can add links, icons, links with icons, search bars, and brand text.

See below our simple <Code>Navbar</Code> example that you can use in your Tailwind CSS and React project.

<br />

Examples on this page are using <Code><a href="https://heroicons.com/" target="_blank">@heroicons/react</a></Code> make sure you have installed it.

```bash
npm i @heroicons/react
```

## Navbar Props
</DocsTitle>

The following props are available for navbar component. These are the custom props
that we've added for the navbar component and you can use all the other
native props as well.

| Attribute   | Type                      | Description                                                               | Default                                |
| ----------- | ------------------------- | ------------------------------------------------------------------------- | -------------------------------------- |
| `variant`   | [Variant](#types-variant) | Change navbar variant                                                     | <Code>filled</Code>                    |
| `color`     | [Color](#types-color)     | Change navbar color                                                       | <Code>white</Code>                     |
| `shadow`    | <Code>boolean</Code>      | Add box-shadow for navbar                                                 | <Code>true</Code>                      |
| `blurred`   | <Code>boolean</Code>      | Add blur backdrop effect for navbar and makes it's background color white | <Code>true</Code>                      |
| `fullWidth` | <Code>boolean</Code>      | Change navbar to a block level element and remove its border radius       | <Code>false</Code>                     |
| `className` | <Code>string</Code>       | Add custom className for navbar                                           | `''`                                   |
| `children`  | <Code>node</Code>         | Add content for navbar                                                    | No default value it's a required prop. |

<DocsTitle href="navbar-theme">
## Navbar Theme
</DocsTitle>

Learn how to customize the theme and styles for navbar component, the theme object for navbar component has three main objects:

**A.** The <Code>defaultProps</Code> object for setting up the default value for props of navbar component.<br />
**B.** The <Code>valid</Code> object for customizing the valid values for navbar component props.<br />
**C.** The <Code>styles</Code> object for customizing the theme and styles of navbar component.<br />

You can customize the theme and styles of navbar component by adding Tailwind CSS classes as key paired values for objects.

<br />
<br />

```ts
interface NavbarStylesType {
```
