# Speedy Boilerplate - Testimonial

The testimonial component is a section that displays the feedbacks of the customers, it's an important part of a website where is shows the credibility of the product or service.

<br />
<br />

<DocsTitle href="testimonial-1">
## Testimonial 1
</DocsTitle>

The component is located in the <Code>components/testimonial/testimonial-1.tsx</Code> file and below is the code snippet of how to use the testimonial component.

<CodePreview noPadding component={<Image src="/boilerplate/testimonial.png" alt="testimonial" width={1024} height={1024} className="w-full" />}>
```tsx
import { Testimonial1 } from "@/components";

export default function Page() {
  return <Testimonial1 />
}
```
</CodePreview>
