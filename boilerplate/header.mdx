# Speedy Boilerplate - Header

The header component is the most important part of the website, it contains the navigation links and the logo of the website. 

<br />
<br />

<DocsTitle href="header-1">
## Header 1
</DocsTitle>

The component is located in the <Code>components/header/header-1.tsx</Code> file and below is the code snippet of how to use the header component.

<CodePreview noPadding component={<Image src="/boilerplate/header.png" alt="header" width={1024} height={1024} className="w-full" />}>
```tsx
import { Header1 } from "@/components";

export default function Page() {
  return <Header1 />
}
```
</CodePreview>
