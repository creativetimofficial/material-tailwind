# Speedy Boilerplate - Footer

The footer component is used to display the footer of the website, like the company name, contact information, social media links, etc.

<br />
<br />

<DocsTitle href="footer-1">
## Footer 1
</DocsTitle>

The component is located in the <Code>components/footer/footer-1.tsx</Code> file and below is the code snippet of how to use the footer component.

<CodePreview noPadding component={<Image src="/boilerplate/footer.png" alt="footer" width={1024} height={1024} className="w-full" />}>
```tsx
import { Footer1 } from "@/components";

export default function Page() {
  return <Footer1 />
}
```
</CodePreview>
