# Speedy Boilerplate - Stats

The stats component is used to display the statistics of the website, like the number of users, the number of downloads, happy customers, etc.


<br />
<br />

<DocsTitle href="stats-1">
## Stats 1
</DocsTitle>

The component is located in the <Code>components/stats/stats-1.tsx</Code> file and below is the code snippet of how to use the stats component.

<CodePreview noPadding component={<Image src="/boilerplate/stats.png" alt="stats" width={1024} height={1024} className="w-full" />}>
```tsx
import { Stats1 } from "@/components";

export default function Page() {
  return <Stats1 />
}
```
</CodePreview>
